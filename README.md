# StayHealthy Frontend

This is a healthcare frontend application developed using React.js.

## Features
- User Login
- User Registration
- Appointment Booking
- Doctor Search
- Reviews
- Profile Management

## Technologies Used
- React.js
- React Router DOM

## Setup Instructions

1. Clone the repository

2. Install dependencies

npm install

3. Start the application

npm start

4. Open browser at:

http://localhost:3000
